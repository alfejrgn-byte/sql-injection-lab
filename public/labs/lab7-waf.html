Lab 7: WAF 우회 ★★★★★

난이도별 필터 규칙을 파악하고, 차단당하지 않는 형태로 검색 쿼리를 변형해 보세요.

미완료
미션

WAF 필터가 특정 SQL 키워드를 차단합니다. 필터를 우회하여 데이터를 추출하세요.

입력

검색어는 사용자명과 이메일에 동시에 반영됩니다. 난이도에 따라 필터 규칙이 달라집니다.

결과

차단되면 WAF 경고가, 통과하면 사용자 검색 결과가 표시됩니다.

기본 검색 결과를 불러오는 중입니다.
실행 쿼리는 여기에 표시됩니다.

힌트

FLAG 입력

필터를 우회해 원하는 데이터를 찾았다면 FLAG를 제출하세요.
