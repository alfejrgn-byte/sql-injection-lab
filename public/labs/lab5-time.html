Lab 5: Time 기반 Blind ★★★★☆

응답 본문이 같더라도 시간을 보면 내부 조건의 참/거짓을 구분할 수 있습니다.

미완료
미션

응답 내용은 항상 동일합니다. 응답 시간 차이로 데이터를 추출하세요. delay_if_true: 뒤에 SQL 조건을 넣어보세요.

입력

일반 ID 조회처럼 보이지만, 특정 접두사를 사용하면 서버가 조건식의 참/거짓에 따라 지연합니다.

결과

본문 메시지는 바뀌지 않습니다. 응답 시간과 막대 길이만 집중해서 보세요.
