Lab 6: Stacked Queries ★★★★☆

문자열 안에 한 문장만 들어간다고 가정한 INSERT 구문을 깨고, 추가 SQL을 이어 실행해 보세요.

미완료
미션

세미콜론(;)으로 추가 SQL 문을 실행할 수 있습니다. INSERT 뒤에 SELECT나 UPDATE를 추가해보세요.

입력

로그 제목은 작은따옴표로 감싸져 INSERT 문에 들어갑니다. 문자열을 닫고 새 문장을 이어 보세요.

결과

기본적으로는 admin_logs 갱신 결과가, 추가 문장이 실행되면 별도 결과 세트가 함께 표시됩니다.

아직 로그를 저장하지 않았습니다.
실행 쿼리는 여기에 표시됩니다.

admin_logs

추가 실행 결과

힌트

FLAG 입력

추가 쿼리 실행에 성공했다면 FLAG를 제출하세요.
